# 264.Ugly-Number-II

所有醜數是由 2、3、5 三條醜數鏈表依序合併出來的

```
1 -> 1*2 -> 2*2 -> 3*2 -> 4*2 -> 5*2 -> 6*2 -> 8*2 ...
1 -> 1*3 -> 2*3 -> 3*3 -> 4*3 -> 5*3 -> 6*3 -> 8*3 ...
1 -> 1*5 -> 2*5 -> 3*5 -> 4*5 -> 5*5 -> 6*5 -> 8*5 ...
```

求第 n 個醜數要合併三條鏈表，類似於 #23 用 3 根指針依序推進鏈表元素

```
if (cur==ret[i]*2) i++;
if (cur==ret[j]*3) j++;
if (cur==ret[k]*5) k++;
```
