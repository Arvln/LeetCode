# 336.Palindrome-Pairs

直覺的解法是N^2，按照數據量會超過1e6將會TLE，思路會往遍歷一遍的方向思考。字串拼接起來是回文代表較長字串可以拆分兩部分，一部分是回文串、另一部分反轉後的字串將出現在字典中。

```
X X X X X

[Y Y Y] X X
X X [Y Y Y]
```

因此任意分割字串，若該字串前半或後半段是回文串，再考慮另一半反轉後是否出現在字典中。由於配對可能出現重複，因此要先將字典內單詞按長度排序再在長度相較當前字串小的字串中找配對。

```
"dcba"+"abcd" "abcd"+"dcba"
"abcd"+"dcba" "dcba"+"abcd"
```
